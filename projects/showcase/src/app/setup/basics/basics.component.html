Initial Setup

To install the library, run: 
npm install ngx-strong-router@latest
The first step to setting up strong routing is to create a targets enum, often simply called Navigation Targtes. This enum should have a member for each route in the application that can be navigated to. For example, the targets for this site are: navigation-targets.enum.ts 
export enum NavigationTargets {{'{'}}
    Home,
    TheoryHome,
    TheoryMagicUrls,
    TheoryRefactoringMishaps,
    TheoryNamedRoutes,
    TheoryStrongRoutes,
    SetupHome,
    SetupBasics,
    SetupLazyLoading,
    SetupNavigation,
    Api
}

Once the targets enum is created, the applcation routes need to be changed from Routes to {{'StrongRoutes\'}}. That allows eagerly loaded routes to use the navigationTarget field. app-routing.module.ts 
const routes: {{'StrongRoutes\'}}[] = [
{{'{'}}
    path: '',
    pathMatch: 'full',
    component: HomeComponent,
    navigationTarget: NavigationTargets.Home
},
{{'{'}}
    path: 'api',
    loadComponent: () => import('./api/api.component').then(c => c.ApiComponent),
    navigationTarget: NavigationTargets.Api
}];
