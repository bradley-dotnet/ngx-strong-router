Strong Navigation

To create a link in a template, use the ngxStrongRouterLink directive instead of routerLink. To import the directive, use: 
import {{'{'}} StrongRouterLink } from 'ngx-strong-router';

The directive's input is a NavigationTarget. Additionally, route parameters can be set as an object through the routeParams input, or navigate relative to the current route with the navigateRelative input. Altogether, it could look something like: linking.component.html 
{{'\'}}
   ViewDetails
{{'\'}}

To navigate within a component or service, inject a StrongRouter instead of a Router. Then call navigateTo or navigateRelativeTo. If you need a link instead, use generateLinkTo or generateLinkRelativeTo. Each also takes the same route parameters object as the directive, as well as the NavigationExtras from the standard Router. To generate the same link as the example above: linking.component.ts 
import {{'{'}} StrongRouter } from 'ngx-strong-router';

@Component() // Component metadata
export class LinkingComponent implements OnInit {{'{'}}
    @Input() item: Item;
    public link: string;
    constructor(private router: StrongRouter) {{'{'}}
    }

    public ngOnInit(): void {{'{'}}
        this.link = router.generateLinkTo(NavigationTargets.List, {{'{'}} itemId: this.item.id });
    }
}
