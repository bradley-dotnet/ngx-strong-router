StrongRouter

Import

import {{'{'}} StrongRouter } from 'ngx-strong-router';

Usage

Use in components or services to replace Router when navigating around the application. Also can generate links to any point in the application.

Methods

navigateTo

Navigate directly to a target.

generateLinkTo

Generate a link to a target.

navigateRelativeTo

Navigate to a target that is a child of the current route.

generateLinkRelativeTo

Generate a link to a target that is a child of the current route.
